# New Command(.bms)
## `#BASE 62` (From Beatoraja 0.8.8)
Specifies that this is base 62 bms, not base 36 bms. This extend key sound limit from 1295 to 3843

## `#SCROLLxx nnn`, `#xxxSC` (From Bemuse v1.0.0-beta.18)
* 260615
Change note speed without `#BPM`. ([Related Article](https://note.com/numuther/n/n57bf895e7969))
